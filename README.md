### Cara menjalankan project di localhost:

> Pastikan telah menginstall php, composer dan nodeJS

-   Buka terminal lalu arahkan ke directory untuk menyimpan project

-   Pull dari repository
    ```sh
    git pull
    ```
-   Install vendor
    ```sh
    composer install
    ```
-   Install node_modules
    ```sh
    npm install
    ```
-   Jalankan project
    ```sh
    php artisan serve
    ```
